tuple

发表于2025-05-05|更新于2025-05-05|数据结构
|浏览量:

在 C++ 中，std::tuple 是一个非常强大的多类型容器，它可以存储多个不同类型的值（类似结构体），但不需要你自定义类型。它非常适合临时存储一组相关数据。

🧱 一、基本概念

✅ tuple 是什么？

  • tuple 是 标准库模板类，位于 <tuple> 头文件中。
  • 可以理解为 “元组”，是一个固定大小的有序集合，每个元素可以是不同类型。
  • 支持任意数量的元素（C++11 起支持最多 100 个字段）。

⚠️ 八、注意事项

事项 说明
不可变长度 tuple 的长度在编译时就确定了
访问效率 get<N>(t) 是常数时间复杂度
类型安全 每个元素类型必须在编译时确定
索引越界 编译时报错（如果使用常量表达式索引）
可读性差 如果字段太多或没有命名，容易混乱（推荐用 struct 替代）

📌 九、与 pair 和 struct 的对比

类型 支持字段数 是否命名字段 适用场景
pair<T1, T2> 2 个 否 临时配对数据
tuple<T1, T2..., Tn> N 个 否 多个不同类型的数据组合
struct { T1 a; T2 b; ... } N 个 是 数据有明确含义时推荐使用

✅ 十、总结一句话

std::tuple 是一种轻量级的多类型容器，适合快速打包多个值，但不适合长期维护或字段很多的情况，建议在这种情况下使用 struct。
